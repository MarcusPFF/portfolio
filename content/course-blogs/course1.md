I denne uge, lavede jeg min portfolio hjemmeside, som du ser lige nu. Jeg brugte Next.js og Tailwind CSS til at lave den.

Flow:

1. Startede med create-next-app og fik et tomt projekt op at køre.
2. Valgte Next.js fordi jeg ville lære React ordentligt, og det giver god struktur fra starten.
3. Tailwind kom på fra dag ét. Jeg gider ikke skrive CSS fra bunden, og Tailwind gør det hurtigt at prøve ting af.
4. Skitserede siden: hero øverst, så projekter, skills og kontakt nederst.
5. Ville have et glass-morphism look. Backdrop-blur, lidt gennemsigtighed, bløde skygger.
6. Lavede en typewriter-effekt i hero'en der skifter mellem mine roller (Developer, QA, Guide, Adventurer).
7. Brugte længere tid end forventet på mobilvisningen. Endte med en hamburger-knap der dropper en hvid menu ned.
8. Slog View Transitions til, så navigationen mellem sider føles glat.
9. Pushede til GitHub og koblede repoet på Vercel. Hver push bygger automatisk.
10. Købte domænet og pegedes det på Vercel.
ƒ
Det sværeste var faktisk ikke koden. Det var alt det med spacing, fontstørrelser og hvor meget tekst hver sektion skulle have. Det tog tid at lande.
