# Marcus Forsberg — Portfolio

A clean, modern developer portfolio built with **Next.js 16** and **Tailwind CSS v4**, featuring a soft glassmorphism aesthetic.

## Tech Stack

- Next.js 16 (App Router)
- Tailwind CSS v4
- TypeScript

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
